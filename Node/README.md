# Node(express)

express 란?

>  Node.js를 위한 웹 프레임워크

개발 환경

>10.16.0

설치

```bash
$ npm install express
# express 프로젝트의 기본 형식을 간단하게 만들어 주는 모듈
$ npm install express-generator
```

프로젝트 생성 및 실행

```bash
# 프로젝트 생성
$ express [프로젝트 폴더명]
# 폴더 이동
$ cd [프로젝트 폴더명]
# 모듈 설치
$ npm install
# 서버 실행
$ npm start
```
