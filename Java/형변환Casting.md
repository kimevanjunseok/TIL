# 형변환Casting

## 사용법

### 1. String to Integer

```java
String stringNum = "10";

int number = Integer.parseInt(stringNum); // 1번째 방법
int number = Integer.valueOf(stringNum); // 2번째 방법
```

### 2. Integer to String

```java
int number = 10;

String stringNum = String.valueOf(number); // 1번째 방법
String stringNum = Integer.toString(number); // 2번째 방법
```
